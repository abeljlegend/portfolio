About Me

Front-End Developer

A Front-End Developer creates websites and applications using web languages such as HTML, CSS, and JavaScript that allow users to access and interact with the site or app. When you visit a website, the design elements you see were created by a front-end developer. They create user interfaces (UI). UI is the graphical layout of an application that determines what each part of a site or application does and how it will look.

Web Designing

Web designing is the process of planning, conceptualizing, and implementing the plan for designing a website in a way that is functional and offers a good user experience.
